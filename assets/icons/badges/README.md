# Badges Directory

This folder is reserved for custom tech-stack badge SVGs (skill icons).

You can add custom badge SVGs here or use shields.io badges directly in your markdown components.

## Usage

Reference badges in your component markdown files:

```markdown
![Badge Name](./assets/icons/badges/your-badge.svg)
```

## Recommended Sources

- [shields.io](https://shields.io/) — Dynamic badge generation
- [simple-icons](https://simpleicons.org/) — Popular brand SVG icons
- [devicons](https://devicon.dev/) — Developer tool icons
